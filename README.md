|Kemo Badji | Page Services  | feature/services |
<!DOCTYPE html> : Indique au navigateur que vous utilisez la version HTML5. C'est indispensable pour un affichage moderne.
<html lang="fr"> : Définit le français comme langue principale. Cela aide les moteurs de recherche (SEO) et les outils d'accessibilité.
<head> : Cette section contient les informations "invisibles" pour l'utilisateur, mais cruciales pour le navigateur et Google.
<meta charset="UTF-8"> : Permet d'afficher correctement les accents (é, à, ê, etc.).
<meta name="viewport" content="width=device-width, initial-scale=1.0"> : Rend votre site Responsive. Il s'adaptera automatiquement à la taille des écrans de smartphones (très important pour un restaurant à Dakar).
<title>Le Menu</title> : C’est le nom qui s'affiche sur l'onglet du navigateur.
<meta name="description" ...> : C'est le petit texte qui apparaîtra sous le nom de votre site dans les résultats de recherche Google.
<link rel="stylesheet" href="style.css"> : Relie votre code HTML au fichier CSS qui contiendra le design (couleurs, polices, mise en page).
